```c
#include <stdio.h>

int main(void) {
	int n = 5;

	if (n > 3) {
		printf("Number %i is greater than 3.\n", n);
	}
}
```
Printf variables in a string quotes are the arguments passed to printf from left to right.
%s - string vars.
%i - integer vars.

If statements support logical operators such AND and OR.
